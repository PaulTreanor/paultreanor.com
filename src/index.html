CANILEAVETHEHOUSE?

A web app built to help people keep track of what lockdown level their county is in and how that restricts their movements.

The website detects the users locations using their IP address and then displays the lockdown status for their location. The user also has the option to choose their location from a dropdown menu if they want to know about other counties or if the location choosen by the app was wrong.

The website was built using Python Flask and Vue.
