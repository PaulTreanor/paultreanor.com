String Theory is an android app I developed with Colin Gorman to help users learn the guitar on the go.

The app was built using Java and XML within Android Studio and is avaiable for download on the Play Store

CANILEAVETHEHOUSE?

Canileavethehouse is a mobile first webapp built to help people keep track of what lockdown level their county is in and how that restricts their movements.

The user can select their location with the map, and then information regarding that area's level of lockdown is returned.

I used Vue and UIkit to create this webapp.
